Circuit: VA → R1 → node → RB → BJT Base; RC from VCC to Collector; −VS at the bottom of R1. Constraints: R1 ≫ RB.

Power RTL NOT Gate — Power Dissipation

Power = Σ (voltage across source) × (current through source). Trace each current path for each input case.

P0 = (VA − (−VS)) × I1

P1 = (VA − 0.8)·IRB + (0.8 − (−VS))·IR1
      + (0.8 − 0)·IB + (VCC − 0)·IC
  • Case (0): Only R1 path active (VA through RB+R1 to −VS). P0 is very small.
  • Case (1): Multiple paths — base drive (VA→RB→base), R1 path (base→R1→−VS), collector path (VCC→RC→collector). Use KCL to find all currents.
  • Average: Pavg = (P0 + P1) / 2
  • Maximum: Pmax = max(P0, P1) — almost always P1.
When −VS = 0 (grounded): Case (0) power drops dramatically, making the Pmax/Pmin ratio extremely large.
Power RTL NOR Gate — Power Dissipation
P(0,0) = 2 × (VA − (−VS)) × I1

P(1,0) = (VA − 0.8)·IRB + (0.8 − (−VS))·IR1
        + (0.8 − 0)·IB + (VCC − 0)·IRC + (VB − (−VS))·I1,other

P(1,1) = 2 × [(VA − (−VS))·IRB + (VB − 0)·IB] + (VCC − 0)·IRC

For case (1,1): IRC splits equally into both collector currents (symmetric circuit).

Noise Margin Theory & Calculation

Definition: The maximum noise voltage tolerable in a transmission line between logic stages without causing incorrect logic detection.

VNH = VOH − VIH    (high noise margin)
VNL = VIL − VOL    (low noise margin)
VN = min(VNH, VNL)    (overall noise margin)
ThresholdDefinitionHow to Find (RTL)
VOL Max low output voltage Fixed at 0.1 or 0.2 V (BJT saturation, given by manufacturer)
VOH Min high output voltage Usually given (depends on loads). When unloaded, VOH = VCC
VIH Min input recognized as HIGH Marginal condition: βforced = βF (BJT shifts from sat → active). Solve for VIH using IC/IB = βF
VIL Max input recognized as LOW Marginal condition: VBE = 0.5 V (BJT turns on from cut-off). With IB = 0 in cut-off, solve KCL
VIH method: Set IC/IB = βF, where IC = (VCC − VOL)/RC (output is still low). Solve for VIH from the base current equation.

VIL method: Set VBE = 0.5 V. Use KCL with IB = 0 (on the boundary of cut-off). Solve: (VIL − 0.5)/RB = (0.5 − (−VS))/R1, giving VIL.
Fanout Maximum Fanout — RTL

Fanout = number of loads connected to a single driver. Maximum fanout = max loads without logic malfunction.

Case: Driver Output HIGH (Load draws current from driver)

Nmax = floor( Isupply / Iload )

Isupply = (VCC − VOH) / RC
Iload = (VOH − 0.8) / RB,load  (per load BJT in sat)

Case: Driver Output LOW (Load delivers current to driver)

Loads increase IC of driver's BJT, risking exit from saturation. Marginal condition:

βforced = βF
IC,total = IC,driver_alone + N × Iload
Set IC,total / IB = βF → solve for N, then floor(N)

Case: No connection (open circuit between driver & load)

Maximum fanout for this case = ∞ (load doesn't affect driver).

Overall maximum fanout = min(Ncase1, Ncase2, …) across ALL input cases.

Diode Transistor Logic (DTL)

NAND Gate · Power Dissipation · Fanout
Operation DTL NAND Gate

Circuit: Input diodes DA, DB → node P → two forward-bias diodes D1, D2 → Base of BJT Q → R2 to −VS. R1 from VCC to node P. RC from VCC to Collector.

Role of D1, D2: They create a voltage barrier. For Q to turn on, the total VP needed is VBE + VD1 + VD2 = 0.5 + 0.6 + 0.6 = 1.7 V. This prevents spurious turn-on.

Input CaseActive DiodesVPQ StateVY
(0,0), (0,1), (1,0) At least one input diode ON Vlow + 0.7 = 0.9 V < 1.7 V Cut-off (D1, D2, Q all OFF) VCC (HIGH) ✓
(1,1) DA, DB both OFF (reverse biased by high input) 2.2 V (= 0.8 + 0.7 + 0.7) Saturation; VCE = 0.2 V 0.2 V (LOW) ✓
Why not just a resistor instead of D1, D2? A resistor can't reliably block VP = 0.9 V from turning on the transistor. The diodes provide a definitive voltage threshold barrier.

DTL as NOT gate: Keep all inputs except one permanently HIGH: AB = A·1 = Ā (NAND with all-but-one tied high becomes a NOT gate).

Power DTL NAND Gate — Power Dissipation
P(0,0) = P(0,1) = P(1,0) = (VCC − 0.2) × IR1
  [Only R1 active; no collector current]

P(1,1) = (VCC − 0) × IC + (VCC − VB) × IR1
            + (VB − 0) × (IR1 − IR2) + (VB − (−VS)) × IR2

For other cases: IR1 = (VCC − VP)/R1 where VP = Vlow + 0.7 = 0.9 V.

For case (1,1): VP = 2.2 V; ID1 = ID2, and IB = ID1 − IR2.

Average Power: Pavg = (P(0,0)×1 + P(0,1)×1 + P(1,0)×1 + P(1,1)×1) / 4

Fanout DTL NAND Gate — Fanout

When DTL NAND gates are used as loads connected to a DTL driver:

  • Driver output HIGH: The input diode of each load DTL gate will be reverse biased (high voltage at cathode). So there is no connection between driver and load → maximum fanout = ∞.
  • Driver output LOW (case 1,1): Load diodes are forward biased, delivering current to the driver's collector node, increasing IC. Marginal condition: βforced = βF of driver BJT.
IC,driver + N × Iload = βF × IB,driver
Solve for N, take floor(N)

Each load delivers: Iload = (VCC − VP,low_output) / R1,load, where VP,low_output = VOL + 0.7.

Overall maximum fanout = min(∞, Nlowcase) = Nlowcase.

Transistor Transistor Logic (TTL)

NAND Gate · Basic Operation · Power Dissipation
Operation Multi-Emitter BJT

A BJT with multiple separate emitter regions. Each emitter forms its own B-E junction. Key rules:

  • If any one B-E junction is forward biased, the BJT is active (not cut-off).
  • Reverse biased emitters play no role in determining operating mode.
  • In reverse active mode (B-C forward, all B-E reverse): IE1 + IE2 + … + IB = IC, with each IEi = βR · IB.
  • IC = IB(1 + N·βR) for N emitters in reverse active mode.
Operation TTL NAND Gate — Circuit Analysis

Circuit: RB1 from VCC to base of Q1 (multi-emitter). Inputs VA, VB at emitters of Q1. Collector of Q1 → Base of Q2. RC2 from VCC to collector of Q2. Emitter of Q2 → RE2 → Base of Q3. RC3 from VCC to collector of Q3. Output VY at collector of Q3.

CaseQ1Q2Q3VYKey Voltages
(0,0) Saturation Cut-off Cut-off VCC (HIGH) VB1 = Vlow + 0.8; VC1 = Vlow + 0.2; VB2 ≈ 0.4 V → Q2 off
(0,1) or (1,0) Saturation Cut-off Cut-off VCC (HIGH) The low-input emitter forward biases; same logic as (0,0)
(1,1) Reverse Active Saturation Saturation 0.2 V (LOW) VB1 = 2.3 V; VB2 = VC1 = 1.6 V; VE2 = VB3 = 0.8 V
Why Reverse Active for Q1 in case (1,1)? Both B-E junctions are reverse biased (high inputs). The B-C junction must be forward biased to carry current to Q2's base. Hence: VBC1 = 0.7 V, VB1 = VC1 + 0.7 = 1.6 + 0.7 = 2.3 V.

Solving Voltages in case (1,1) — step by step

  1. Assume Q3 in saturation → VB3 = 0.8 V
  2. VE2 = VB3 = 0.8 V → VB2 = VE2 + VBE2 = 0.8 + 0.8 = 1.6 V
  3. VC1 = VB2 = 1.6 V
  4. VBC1 = 0.7 V → VB1 = VC1 + 0.7 = 2.3 V
  5. IB1 = i1 = (VCC − VB1) / RB1
  6. IEA = IEB = βR × IB1 (current from emitters toward external circuit)
  7. IC1 = IB2 = IEA + IEB + IB1
Power TTL NAND Gate — Power Dissipation
P(0,0) = P(0,1) = P(1,0) = (VCC − Vlow) × IRB1
[Q2 & Q3 in cut-off; only RB1 path active]

P(1,1) = (VA − 0)·IEA + (VB − 0)·IEB
        + (VCC − 0)·[IRB1 + IRC2 + IRC3]
  • In case (0,0)/(0,1)/(1,0): Only IRB1 flows (through RB1 into Q1's base, and out through the forward-biased B-E junction to the low-input terminal).
  • In case (1,1): Current flows from both emitter inputs (VA, VB → emitters → external), and from VCC through RB1, RC2, and RC3.
Pavg = (3 × Plow_cases + P(1,1)) / 4

NMOS Logic

NOT Gate · NOR Gate · Resistive, Enhancement & Depletion Loads
MOSFET Operating Modes — Summary
ModeCondition (NMOS)IDS Formula
Cut-off VGS < VTN IDS = 0 (open circuit)
Triode (Linear) VGS > VTN and VDS < VOV IDS = Kn[2(VGS − VTN)VDS − VDS²]
Saturation VGS > VTN and VDS > VOV IDS = Kn(VGS − VTN)²
Overdrive voltage: VOV = VGS − VTN
Transition (pinch-off) point: VDS = VOV

PMOS: Use VSG, VSD, VTP (negative for enhancement type); ID = Kp[2(VSG + VTP)VSD − VSD²] (triode) or ID = Kp(VSG + VTP)² (saturation).

ENHANCEMENT NMOS

VTN > 0. No channel at VGS = 0. Gate voltage must exceed VTN to form channel.

DEPLETION NMOS

VTN < 0. Channel exists at VGS = 0. Always ON; gate can reduce or cut off channel.

Operation NMOS Inverter — Resistive Load
Input VANMOS StateVY
Low (0 V) Cut-off (VGS < VTN) VDD (IRD = 0)
High (VDD) Triode (designed so VDS < VOV) Very low (≈ 0.05–0.3 V typically)

Solving for VY in Case (1): Set IDS = IRD. Substitute triode formula, solve the quadratic. Discard the unphysical root (the one where IDS would be negative = current flowing backward).

Kn[2(VGS − VTN)VY − VY²] = (VDD − VY) / RD

Verification: After solving, confirm VY < VOV = VGS − VTN (triode condition).

Transition Point (Triode → Saturation boundary)

At transition: VDS = VOV, and saturation IDS = Kn·VY² (since VDS=VY=VOV). Set equal to IRD:

Kn·VY² = (VDD − VY) / RD
Solve for VY (transition output voltage).
Then: VA,transition = VY + VTN

(0,1) or (1,0) One triode, one cut-off Kn[2(VGS−VTN)VY−VY²] = (VDD−VY)/RD Low (solve quadratic)

Power NMOS Logic — Power Dissipation
P(0) = 0   [Cut-off; IDS = 0, no current flows]
P(1) = VDD × IRD = VDD × (VDD − VY) / RD

Pavg = (P(0) + P(1)) / 2

MOSFET gate current = 0 always. Power dissipation = (VDD − 0) × Isupply for all "on" cases.

Operation Enhancement Type Load

Replace RD with an enhancement NMOS (ML) with its gate and drain shorted (VG = VD).

  • VDS,L = VGS,L → always greater than or equal to VOV,L → always in saturation.
  • VGS,L = VDD − VY (since gate=drain=top node, source=output node).
IDL = KnL(VGS,L − VTN,L)²
= KnL(VDD − VY − VTN,L)²

Case (0): Driver in cut-off → IDD = 0 → IDL = 0 → VDD − VY − VTN,L = 0 → VY = VDD − VTN,L.

Case (1): Set IDL = IDD (driver in triode), solve the resulting quadratic for VY.

Operation Depletion Type Load

Replace RD with a depletion NMOS (ML) with its gate and source shorted (VGS,L = 0). VTN,L < 0.

  • VOV,L = VGS,L − VTN,L = 0 − VTN,L = |VTN,L| > 0.
  • If VSD,L > VOV,L: saturation → IDL = KnL(0 − VTN,L)² = KnL·VTN,L² (constant!).
  • If VSD,L < VOV,L: triode.
Key insight for depletion load: When output VY is expected LOW (driver ON cases), the load ML will be in saturation (high VDS,L), providing a constant current source. When VY is high (driver OFF), ML may be in triode → IDL = 0 → VY = VDD.
Depletion load in saturation:
IDL = KnL(0 − VTN,L)² = KnL × VTN,L²

Note: With Kn' = Kn·(W/L), the effective Kn for each MOSFET = Kn'·(W/L) for that device.

CMOS Logic

VTC · Noise Margin · Logic Implementation
VTC Ideal VTC of CMOS Inverter — 5 Regions

Circuit: PMOS (MP) between VDD and output; NMOS (MN) between output and GND. Both gates connected to input VI.

Region Input Range MN State MP State Output VO
1 0 < VI < VTN Cut-off Triode VDD
2 VTN < VI < VIt Saturation Triode High → decreasing (solve IDN=IDP)
3 VI = VIt Saturation (transition) Saturation (transition) VO falls steeply through VIt
4 VIt < VI < VDD−|VTP| Triode Saturation Low → decreasing (solve IDN=IDP)
5 VDD−|VTP| < VI < VDD Triode Cut-off 0 V
In regions 1 and 5, one transistor is cut-off → no current flows → zero static power dissipation. This is the key advantage of CMOS over NMOS/BJT logic.
Formula Transition Voltage VIt and Critical Points

At VI = VIt: both transistors in saturation simultaneously. Set IDN = IDP:

KN(VIt − VTN)² = KP(VDD − VIt + VTP)²

Taking square roots (both sides positive):

√KN (VIt − VTN) = √KP (VDD − VIt + VTP)
VIt = [√KP(VDD + VTP) + √KN·VTN] / (√KN + √KP)

When KN = KP and |VTP| = VTN:

VIt = VDD/2   (symmetric inverter)

Output Transition Points

VOPt = VIt − VTP   [VO when MP transitions triode→sat]
VONt = VIt − VTN   [VO when MN transitions sat→triode]
Noise Margin CMOS Inverter Noise Margin

The actual (practical) VTC differs from the ideal: the transition region is not perfectly vertical. The inflection points (slope = −1) define VIH and VIL.

NMH = VOH − VIH
NML = VIL − VOL

Finding VIL and VOH (MN in sat, MP in triode)

  1. Write IDN = IDP with MN saturation formula and MP triode formula.
  2. Differentiate both sides with respect to VI.
  3. Set dVO/dVI = −1 (slope condition at inflection point).
  4. This gives a relation: VIL = VOH − (VDD/2) (for KN=KP).
  5. Substitute back into original current equation to find VOH and VIL.

Finding VIH and VOL (MN in triode, MP in sat)

  1. Write IDN = IDP with MN triode formula and MP saturation formula.
  2. Differentiate and set dVO/dVI = −1.
  3. This gives: VOL = VIH − (VDD/2) (for KN=KP).
  4. Substitute back to find VIH and VOL.
Symmetric case (KN=KP, |VTP|=VTN): NMH = NML. The VTC is symmetric about VDD/2. Example: VDD=5V, KN=KP=50µA/V², VTN=1V → VOH=4.625V, VIL=2.125V, VIH=2.875V, VOL=0.375V → NMH=NML=1.75V.
Implementation CMOS Logic Circuit Design Rules
  1. High input: turns ON NMOS, turns OFF PMOS. Low input does the opposite.
  2. ON MOSFET = short circuit (drain–source); OFF MOSFET = open circuit.
  3. Conduction Complement: NMOS network in series ↔ PMOS network in parallel (and vice versa). The PMOS network is the exact structural complement of the NMOS network.
  4. Pull-Down Network (PDN): made of NMOS transistors. When output should be LOW, a conducting path to GND exists.
  5. Pull-Up Network (PUN): made of PMOS transistors. When output should be HIGH, a conducting path to VDD exists.
  6. No resistors needed in CMOS — PMOS plays the role of the load.
  7. For non-inverted function Y = f(A,B,…): First design CMOS network for Ȳ, then add a CMOS inverter at output → Y.
  8. One transistor per variable, per network.

Design Procedure

Step 1: Identify if the desired output requires an inversion. If Y = f (not an inverter), design CMOS for Ȳ = complemented form, then invert.

Step 2: Design PDN (NMOS) for Ȳ: series NMOS for AND terms, parallel for OR terms.

Step 3: Apply conduction complement to get PUN (PMOS): swap series↔parallel.

Step 4: Connect PUN between VDD and output; connect PDN between output and GND.

Step 5: If needed, add a CMOS inverter to invert Ȳ and get Y.

RTL NOR Gate — Noise Margin & Fanout

Full derivation methodology
Noise Margin RTL NOR Gate

For a NOR gate driving another NOR gate, the method is identical to the NOT gate. The difference is which transistor (QA or QB) is being analyzed.

Finding VIL (one input; other input = LOW)

Marginal condition: VBE = 0.5 V. With IB = 0 (boundary of cut-off), apply KCL at the base node:

(VIL − 0.5) / RB = (0.5 − (−VS)) / R1

VIL = 0.5 + RB × (0.5 + VS) / R1

Finding VIH (one input; other input = LOW)

Marginal condition: βforced = βF. The active transistor is at the boundary of saturation. IC is set by VCC, RC, and VOL.

IC = (VCC − VOL) / RC
IB = IC / βF

KCL at base: IB = (VIH − 0.8)/RB − (0.8 − (−VS))/R1

Solve for VIH: VIH = 0.8 + RB × [IB + (0.8+VS)/R1]
For case (1,1) with both inputs HIGH: Both BJTs in saturation. If the driven input switches to LOW, only that BJT cuts off; the other keeps output LOW. So VIH found for the single-active-transistor case applies, and the NOR gate is NOT more sensitive to noise in this configuration.
Fanout RTL NOR Gate

Same principle as NOT gate fanout, but the driver NOR gate has two BJTs in parallel sharing RC.

Driver output HIGH → all BJTs cut-off

Each load NOR input draws: Iin,load = (VOH − 0.8) / RB,load per active load transistor
Available current from driver: Iavailable = (VCC − VOH) / RC,driver
Nmax = floor(Iavailable / Iin,load)

Driver output LOW → parallel BJTs in saturation

Load gates feed current into the output node (driver's collectors). Marginal condition: βforced = βF for each driver BJT.

IC,total = IC,driver,alone + N × Iload
where IC,driver,alone = (VCC − VOL) / RC
and IB,each = computed from individual base drive

Condition: IC,total / IB,each = βF; solve for N

Overall max fanout = min of all cases above.

NMOS Logic — Enhancement & Depletion Loads (Complete)

NOT Gate & NOR Gate · All Three Load Types · Full Analysis
Enhancement Load NMOS NOT Gate — Full Analysis

Load: ML (enhancement, gate shorted to drain). Driver: MD (enhancement). Gate of ML = VDD (drain of ML). Source of ML = VY (output).

Load ML — Always in Saturation

VGS,L = VDD − VY
VDS,L = VDD − VY = VGS,L → VDS,L = VGS,L ≥ VOV,L always → Saturation

IDL = KnL(VGS,L − VTN,L)² = KnL(VDD − VY − VTN,L)²

Case (0): VA = 0 V → MD cut-off

IDD = 0 → IDL = 0
KnL(VDD − VY − VTN,L)² = 0
∴ VY = VOH = VDD − VTN,L
Note: VOH = VDD − VTN,L < VDD for enhancement load. This is a disadvantage compared to resistive or depletion loads.

Case (1): VA = VDD → MD in triode (designed)

Set IDL = IDD:

KnL(VDD − VY − VTN,L)² = KnD[2(VDD − VTN,D)VY − VY²]

Rearrange into a quadratic in VY. Let α = KnL/KnD:

(1+α)VY² − [2(VDD−VTN,D) + 2α(VDD−VTN,L)]VY + α(VDD−VTN,L)² = 0

Solve using quadratic formula; take the smaller root (VOL).

Verification: Confirm VY < VOV,D = VDD − VTN,D (MD in triode ✓).

Power Dissipation — Enhancement Load NOT Gate

P(0) = VDD × IDL(VY=VOH) = 0   [IDL=0 when VY=VOH]

P(1) = VDD × ID   [ID = IDL = IDD; use either formula to find ID]

Pavg = (0 + P(1)) / 2 = P(1) / 2
Depletion Load NMOS NOT Gate — Full Analysis

Load: ML (depletion, gate shorted to source → VGS,L = 0, VTN,L < 0). Driver: MD (enhancement).

Load ML operating mode check

VOV,L = VGS,L − VTN,L = 0 − VTN,L = |VTN,L| (positive)

If VDS,L = VDD − VY > |VTN,L| → Saturation → IDL = KnL(0 − VTN,L)² = KnL·VTN,L² (constant)
If VDS,L < |VTN,L| → Triode

Case (0): VA = 0 → MD cut-off

IDD = 0 → IDL = 0
If ML in triode: IDL = KnL[2(0−VTN,L)(VDD−VY) − (VDD−VY)²] = 0
∴ VDD − VY = 0 → VY = VOH = VDD
Key advantage: Depletion load achieves VOH = VDD (full rail), unlike enhancement load which loses VTN,L.

Case (1): VA = VDD → MD in triode; ML in saturation

IDL = KnL × VTN,L²   (constant; VDS,L large → saturation)

Set equal to IDD (MD in triode):
KnL·VTN,L² = KnD[2(VDD−VTN,D)VY − VY²]

Solve quadratic for VY = VOL.

This gives a very small VOL because the constant current KnL·VTN,L² is small relative to MD's drive strength — one reason depletion-load NMOS is preferred over resistive load.

Power Dissipation — Depletion Load NOT Gate

P(0) = 0   [No current flows; IDL=0 when MD is off and ML forced to triode]

P(1) = VDD × IDL = VDD × KnL × VTN,L²

Pavg = P(1) / 2

CMOS Noise Margin — Complete Derivation

Step-by-step differentiation to find VIL, VOH, VIH, VOL
Setup: CMOS Inverter Current Balance

At any operating point, IDN = IDP (no gate current in MOSFETs; KCL at output node).

NMOS: VGS,N = VI, VDS,N = VO
PMOS: VSG,P = VDD−VI, VSD,P = VDD−VO
Region 2 MN Saturation, MP Triode → Find VIL and VOH

Step 1 — Write current equations

MN (Saturation): IDN = KN(VI − VTN)²

MP (Triode): IDP = KP[2(VDD−VI+VTP)(VDD−VO) − (VDD−VO)²]

Set IDN = IDP … (*)

Step 2 — Differentiate (*) with respect to VI

Left side: d/dVI[KN(VI−VTN)²] = 2KN(VI−VTN)

Right side (apply chain rule, VO is a function of VI):

d/dVI[KP(2(VDD−VI+VTP)(VDD−VO) − (VDD−VO)²)]

= KP[−2(VDD−VO) + 2(VDD−VI+VTP)(−dVO/dVI) − 2(VDD−VO)(−dVO/dVI)]

= KP[−2(VDD−VO) + 2(dVO/dVI)(VO − VDD − (VDD−VI+VTP))]

Let m = dVO/dVI

Step 3 — Set m = dVO/dVI = −1 (inflection point condition)

2KN(VI−VTN) = KP[−2(VDD−VO) + 2(−1)(VO−VDD−(VDD−VI+VTP))]

2KN(VI−VTN) = KP[−2(VDD−VO) − 2VO + 2VDD + 2(VDD−VI+VTP)]

2KN(VI−VTN) = 2KP(VDD−VI+VTP)

This gives the relationship between VI = VIL and VO = VOH:

At slope = −1 (Region 2 inflection):

KN(VIL − VTN) = KP(VDD − VIL + VTP)

∴ VIL = [KN·VTN − KP(VDD+VTP)] / [−(KN+KP)]

For symmetric case (KN=KP=K, VTN=|VTP|=VT):
VIL = VDD/2 − VT/2 + VT/2 → substitute into (*) to get VOH

Step 4 — Find VOH by substituting VIL back into (*)

KN(VIL−VTN)² = KP[2(VDD−VIL+VTP)(VDD−VOH) − (VDD−VOH)²]

Let: a = KN(VIL−VTN)² / KP  (known, since VIL found above)
Let: x = VDD − VOH, b = VDD−VIL+VTP
Then: a = 2bx − x² → x² − 2bx + a = 0
Solve: x = b − √(b²−a)  (take smaller root since VOH is large → x is small)
VOH = VDD − x
Region 4 MN Triode, MP Saturation → Find VIH and VOL

Step 1 — Write current equations

MN (Triode): IDN = KN[2(VI−VTN)VO − VO²]

MP (Saturation): IDP = KP(VDD−VI+VTP)²

Set IDN = IDP … (**)

Step 2 — Differentiate (**) with respect to VI

d/dVI[KN(2(VI−VTN)VO−VO²)] = d/dVI[KP(VDD−VI+VTP)²]

KN[2VO + 2(VI−VTN)m − 2VO·m] = KP·2(VDD−VI+VTP)(−1)

where m = dVO/dVI

Step 3 — Set m = −1

KN[2VO + 2(VI−VTN)(−1) − 2VO(−1)] = −2KP(VDD−VI+VTP)

KN[2VO − 2(VI−VTN) + 2VO] = −2KP(VDD−VI+VTP)

KN[4VO − 2(VI−VTN)] = −2KP(VDD−VI+VTP)
At slope = −1 (Region 4 inflection):

2KN(VIH−VTN−2VOL) = KP(VDD−VIH+VTP)

Symmetric case (KN=KP=K):
2K(VIH−VTN−2VOL) = K(VDD−VIH+VTP)
This relates VIH and VOL; substitute back into (**) for the second equation to solve both.

Step 4 — Find VOL by substituting VIH back into (**)

KN[2(VIH−VTN)VOL − VOL²] = KP(VDD−VIH+VTP)²

Let: c = KP(VDD−VIH+VTP)² / KN  (known)
d = VIH − VTN
Then: 2d·VOL − VOL² = c → VOL² − 2d·VOL + c = 0
Solve: VOL = d − √(d²−c)  (take smaller root)
Summary Symmetric Case Results (KN=KP, |VTP|=VTN=VT)
VIt = VDD/2
VIL = (3VDD/8) + VT/4   [exact value depends on K and VT; derive per above]
VIH = VDD − VIL   [by symmetry]
VOH = VDD − VOL   [by symmetry]
NMH = NML
NM = VOH − VIH = VIL − VOL
General approach for any CMOS problem:
1. Identify which region (2 or 4) you are solving for.
2. Write the IDN=IDP equation with appropriate mode formulas.
3. Differentiate both sides with respect to VI.
4. Substitute dVO/dVI = −1; solve for VI (= VIL or VIH).
5. Substitute back into original equation; solve for VO (= VOH or VOL).
6. Compute NMH = VOH − VIH and NML = VIL − VOL.
VTC CMOS Regions 2 & 4 — Solving for VO at any VI

Region 2 (VTN < VI < VIt): MN Sat, MP Triode

IDN = IDP:
KN(VI−VTN)² = KP[2(VDD−VI+VTP)(VDD−VO) − (VDD−VO)²]

Let x = VDD − VO, A = VDD−VI+VTP:
KN(VI−VTN)² / KP = 2Ax − x²
x² − 2Ax + [KN(VI−VTN)²/KP] = 0
Solve for x → VO = VDD − x (take smaller x → larger VO since still HIGH)

Region 4 (VIt < VI < VDD−|VTP|): MN Triode, MP Sat

KN[2(VI−VTN)VO − VO²] = KP(VDD−VI+VTP)²

Let B = VI−VTN, C = KP(VDD−VI+VTP)²/KN:
2B·VO − VO² = C
VO² − 2B·VO + C = 0
Solve for VO (take smaller root → VO is LOW in this region)

DTL NAND Gate — Noise Margin

Threshold voltages for the DTL family
Noise Margin DTL NAND Gate

The diodes D1, D2 shift the threshold significantly compared to RTL.

Finding VIL — Marginal cut-off condition

BJT is at the boundary of cut-off: VBE = 0.5 V. This means VB = 0.5 V (base of Q). Then VP = VB + VD2 + VD1 = 0.5 + 0.7 + 0.7 = 1.9 V (marginal conduction through D1, D2). But for the input diode DA to be still on (keeping VP clamped to VIL + 0.7):

VP = VIL + 0.7 V   (input diode DA still conducting)

Marginal condition: VP = 0.5 + 0.7 + 0.7 = 1.9 V
∴ VIL = 1.9 − 0.7 = 1.2 V

Finding VIH — Marginal saturation condition

BJT at boundary between active and saturation: βforced = βF. As input rises, input diode DA becomes reverse biased when VIH > VP − 0.7 = 2.2 − 0.7 = 1.5 V. Once reverse biased, VIH can be computed using the base current equation:

For VA ≥ 1.5 V: input diode DA is reverse biased; VP set by VCC, R1, and base drive.

IC = (VCC − VOL) / RC   (output low = BJT in saturation)
IB = IC / βF   (marginal condition)

KCL at node P: IR1 = ID1 = ID2+IB+IR2
Using VP = 2.2 V (all junctions in normal conduction):
IR1 = (VCC − 2.2)/R1
IR2 = (0.8 − (−VS))/R2
IB = IR1 − IR2

At marginal condition: set computed IB = IC/βF → this gives VIH implicitly. Since VA=VIH does not appear directly when DA is reverse biased, VIH is effectively the voltage at which DA just becomes reverse biased ≈ VP − 0.7 = 1.5 V.
NML = VIL − VOL = 1.2 − 0.2 = 1.0 V   (typical)
NMH = VOH − VIH = 5.0 − 1.5 = 3.5 V   (typical, VCC=5V)

DTL has excellent noise margins (much better than RTL) due to diode voltage threshold.

CMOS vs. NMOS — Key Differences

CMOS ADVANTAGES

  • Zero static power dissipation (both transistors never ON simultaneously)
  • VOH = VDD, VOL = 0 (full rail-to-rail swing)
  • NM ≈ VDD/2 (symmetric and large)
  • No load resistors needed
  • Scales well with VDD

CMOS DISADVANTAGES

  • Requires both NMOS and PMOS → more complex fabrication
  • Dynamic power: P = C·VDD²·f during switching
  • Short-circuit current flows during switching (both briefly ON)
  • Larger gate area than NMOS-only designs
